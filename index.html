FRS BASED PORTABLE ATTENDENCE SYSTEM

Guide : Dr. Tapasil, Dr. Naveen Kumar, Dr. Shekar

The project is aimed at designing FRS(Fingerprint Recognition System) based portable attendance system which will manage the student attendance at IIITDM, Kancheepuram.

The Fingerprint Attendance System using Raspberry pi is based on feature matching algorithm and compare with previously stored templates of fingerprint against user’s fingerprint for authentication.

It consists of three phases namely; enrolment, attendance, synchronisation of data. An Attendance portal has also been developed as a part of this project for faculty to view the generated reports.
